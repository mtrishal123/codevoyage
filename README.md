# CodeVoyage

AI-powered tech stack migration planning using multi-agent analysis.

## What it does

Analyzes any GitHub repository and creates:
- Current tech stack analysis
- Migration research and best practices
- Effort and cost estimates
- Complete migration roadmap

## Built with

- Subconscious AI Platform (Multi-agent orchestration)
- TypeScript
- Node.js
